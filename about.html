About this project

This is a project was an by the idea of @Julian Seeholzer and was realized by @Laurin Seeholzer in august 2021. It was a part of learning how to create physics simulations with javascript in the browser. Therefor an extreamly powerfull library called matter.js was used to create the simulation of the brachistochrone curve and the falling balls.
